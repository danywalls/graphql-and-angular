# Building Apps with Angular and GraphQL: From REST to GraphQL

## Project Summary
Learn how to configure and set up a GraphQL server to allow clients to request exact data in an Angular app, transitioning from REST to GraphQL efficiently.

## How to Run
### Installation
1. `npm install` 

### Running the Server
1. `npm run dev` or `node index.js` to start the GraphQL server.
